This is a Frutiger aero theme for Cinnamon DE based on B00merang Windows 7 theme: https://github.com/B00merang-Project/windows-7. I have also borrowed some code from Soul-Dark theme for the new Cinnamon menu: https://github.com/linuxmint/cinnamon-spices-themes/tree/master/Soul-Dark. This theme is best used with the Blur Cinnamon extension and it has been only tested on Cinnamon Desktop environment. Personally I have paired this theme with the Mint-X-Yellow icons, DMZ-White mouse cursor, CinnVIIStarkMenu + Clean Show Desktop applets, for an authentic 2009 feel, along with the B00merang Windows 7 and DarkIce window border themes.
