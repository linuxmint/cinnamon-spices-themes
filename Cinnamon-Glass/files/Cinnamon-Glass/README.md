Notes:
------
- This theme is compatible with both the new and classic Cinnamon menus.
- I highly recommend using the Blur Cinnamon extension with this theme for a frosted glass look and better text readability.
- I recommend disabling symbolic icons for categories and system buttons (with Configurable menu applet) for a less flatter look.

How did I achieve the look on the screenshot?
---------------------------------------------
- Used Mint-X-Yellow icons. You can also use Mint-X icons & variants and also Obsidian icon theme for a similiar look.
- Used the "3D Render" wallpaper by "Rawel Czerwinski" for Linux Mint.
